# 學校活動：兒童畫魚水族箱投影系統 — 架構設計

> 狀態：待使用者確認（2026-06-11 提出）
> 開發進度請看 `docs/PROGRESS.md`（接手開發必讀）

## 1. 活動流程（使用情境）

1. 小朋友在紙上畫魚
2. 工作人員/家長用手機拍照（最多同時 5+ 台手機）
3. 手機瀏覽器掃 QR Code 開上傳頁 → 選照片 → 框選魚 → 送出
4. 系統自動去背（白紙背景移除）→ 魚立即「游進」投影幕
5. 投影幕：背景圖自動輪播 + 26 隻魚輪流登場，有水底漂浮感

## 2. 技術選型

| 項目 | 選擇 | 理由 |
|---|---|---|
| 伺服器 | Node.js + Express | 單一筆電即可跑，無需資料庫 |
| 即時推送 | WebSocket (ws) | 新魚上傳後 1 秒內出現在投影幕 |
| 投影動畫 | PixiJS v8 (WebGL) | 60fps、濾鏡（位移/模糊）做水波與光影 |
| 圖片處理 | sharp（伺服器端） | 去背、裁切、縮圖、轉 PNG 透明 |
| 儲存 | 檔案系統 + JSON | `data/fish/`、`data/backgrounds/`、`data/state.json` |
| 網路 | 校內 Wi-Fi 或筆電開熱點 | **完全離線可用**，所有 library 打包本地 |

不用資料庫、不用雲端 — 活動現場斷網也能跑。

## 3. 系統組成（三個頁面 + 一個伺服器）

```
┌─ 筆電（接投影機）──────────────────────────┐
│  Node.js server (port 3000)                 │
│  ├── GET /          → 投影頁（全螢幕水族箱）  │
│  ├── GET /upload    → 手機上傳頁（QR 進入）   │
│  ├── GET /admin     → 控制台                 │
│  ├── POST /api/fish → 上傳照片               │
│  ├── WebSocket      → 推播新魚/設定變更       │
│  └── data/          → 魚、背景、設定          │
└─────────────────────────────────────────────┘
        ▲ 同一個 Wi-Fi
   📱×5+ 手機（瀏覽器，免裝 App）
```

### 3.1 投影頁 `/`（核心）

- **背景輪播**：自動每 N 秒切換（admin 可設），交叉淡入淡出 + 緩慢 Ken Burns 縮放平移
- **魚的游動**：
  - 水平游動 + 正弦波上下擺動 + 隨機微漂移（水流感）
  - 尾巴擺動：對魚圖做週期性 skew/網格波動變形
  - 轉向時水平翻面；偶爾加速衝刺、偶爾停留漂浮
  - 深度分層：遠的魚較小、較暗、微模糊；近的大而清晰 → 立體感
- **魚的輪替（26 隻）**：
  - 同時在場上限 M 隻（預設 10，admin 可調）
  - 每隻游 T 秒後自然游出畫面邊緣，佇列中下一隻游入 → 公平輪替，全部都會出現
  - **新上傳的魚插隊立刻登場**：入場特效（氣泡爆出）+ 顯示名字橫幅「○○○的魚來了！」
- **水底氛圍（質感重點）**：
  - 上方斜射光束（god rays，緩慢搖曳）
  - 上升氣泡粒子、漂浮微粒（浮游生物感）
  - caustics 水波光紋疊加層
  - 整體藍色調 vignette；魚加極輕微藍色 tint 融入水中
  - 可選：環境水聲/泡泡音效開關

### 3.2 手機上傳頁 `/upload`

- 手機瀏覽器開啟，免安裝 App，介面大按鈕、中文
- 流程：選/拍照片 → 畫面上拖框裁切魚 → 填小朋友名字（選填）→ 送出
- 上傳後顯示「你的魚游進去了！」確認畫面
- 伺服器自動處理：白紙背景去背（亮度/色彩 flood-fill）、去陰影、裁邊、縮圖、存 PNG 透明
- 若手機已用 App 去背好的 PNG，直接通過不破壞

### 3.3 控制台 `/admin`

- 背景管理：上傳/刪除背景圖、設定輪播秒數、手動切換
- 魚管理：縮圖清單、隱藏/刪除（畫不適當的可即時下架）、「主打這隻」按鈕（讓某隻游到中央放大）
- 參數：同場魚數上限、游速、每隻在場秒數
- 暫停/播放、顯示上傳頁 QR Code（投影或列印用）
- 全部設定即時生效（WebSocket 推到投影頁）

## 5. 讓活動更順的建議（已納入設計）

1. **拍照小技巧頁**：上傳頁附提示 — 紙攤平、光線均勻、避免陰影、垂直拍 → 去背成功率大增
2. **活動前測試模式**：內建幾隻範例魚，到場接投影機即可先彩排
3. **QR Code 大圖**：admin 可一鍵顯示全螢幕 QR，列印貼在上傳桌
4. **斷網保險**：筆電自開熱點即可，全離線
5. **不當圖片即時下架**：admin 一鍵隱藏
6. **名字橫幅**：小朋友看到自己名字 + 魚登場，是活動高潮點
7. **投影質感**：1080p 全螢幕、深色背景下光束+氣泡+水紋，建議現場關燈
8. **RUNBOOK.md**：當天操作手冊（開機順序、網路設定、常見問題）
